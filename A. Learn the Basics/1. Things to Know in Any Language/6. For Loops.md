# Lec 1: Things to Know in C++ / Java / Python
# Topic: For Loops

A **for loop** is an **iteration (looping) statement** used to execute a block of code repeatedly for a known number of times.

Instead of writing the same code multiple times, a `for` loop automates repetition.

---

# Why Do We Need Loops?

Suppose you want to print numbers from **1 to 5**.

Without a loop:

```java
System.out.println(1);
System.out.println(2);
System.out.println(3);
System.out.println(4);
System.out.println(5);
```

With a loop:

```java
for(int i = 1; i <= 5; i++){
    System.out.println(i);
}
```

The loop is shorter, cleaner, and scalable.

---

# What is a For Loop?

A `for` loop consists of **three parts**:

```java
for(initialization; condition; update){
    // Code
}
```

### Structure

```java
for(int i = 0; i < n; i++){

}
```

| Part | Purpose |
|------|---------|
| Initialization | Executes once before the loop starts |
| Condition | Checked before every iteration |
| Update | Executes after every iteration |

---

# Flow of Execution

```
          Start
             │
             ▼
     Initialization
             │
             ▼
      Check Condition
       /          \
    True          False
      │             │
      ▼             ▼
 Execute Loop      End
      │
      ▼
 Update Variable
      │
      └──────────────►
```

# Enhanced For Loop (For-Each Loop)

Used to iterate over arrays or collections without using an index.

### Syntax

```java
for(dataType variable : collection){

}
```

Example

```java
int[] arr = {10, 20, 30};

for(int x : arr){
    System.out.println(x);
}
```

Output

```
10
20
30
```

---

# Difference Between Normal and Enhanced For Loop

| Normal For Loop | Enhanced For Loop |
|-----------------|------------------|
| Uses index | No index |
| Can modify elements using index | Read-only traversal (primitives) |
| Works with arrays and collections | Works with arrays and collections |
| More flexible | Simpler syntax |

---

# Nested For Loops

A `for` loop inside another `for` loop.

Example

```java
for(int i = 1; i <= 3; i++){
    for(int j = 1; j <= 2; j++){
        System.out.println(i + " " + j);
    }
}
```

Output

```
1 1
1 2
2 1
2 2
3 1
3 2
```

---

# Pattern Printing Example

```java
for(int i = 1; i <= 4; i++){
    for(int j = 1; j <= i; j++){
        System.out.print("*");
    }
    System.out.println();
}
```

Output

```
*
**
***
****
```

Pattern printing is an excellent way to understand nested loops.

# Infinite For Loop

```java
for(;;){
    System.out.println("Hello");
}
```

Since no condition is provided, the loop never terminates unless a `break` statement is used.

---

# Using `break`

Stops the loop immediately.

```java
for(int i = 1; i <= 10; i++){

    if(i == 5){
        break;
    }

    System.out.println(i);
}
```

Output

```
1
2
3
4
```

---

# Using `continue`

Skips the current iteration and moves to the next one.

```java
for(int i = 1; i <= 5; i++){

    if(i == 3){
        continue;
    }

    System.out.println(i);
}
```

Output

```
1
2
4
5
```

# Common Mistakes

## 1. Off-by-One Error

Wrong

```java
for(int i = 0; i <= arr.length; i++)
```

This causes:

```
ArrayIndexOutOfBoundsException
```

Correct

```java
for(int i = 0; i < arr.length; i++)
```

---

## 2. Forgetting Update

Wrong

```java
for(int i = 0; i < 5;){
    System.out.println(i);
}
```

Infinite loop because `i` never changes.

---

## 3. Wrong Condition

Wrong

```java
for(int i = 10; i <= 1; i--)
```

Loop never executes.

Correct

```java
for(int i = 10; i >= 1; i--)
```

---

## 4. Modifying Loop Variable Incorrectly

```java
for(int i = 0; i < n; i++){
    i++;
}
```

This skips every alternate element.

---

# Competitive Programming Tips

- Most array problems use:

```java
for(int i = 0; i < n; i++)
```

- Use `arr.length` instead of hardcoded sizes.
- Use nested loops carefully because they can increase time complexity.
- Understand the difference between `break` and `continue`.
- Use enhanced `for` loops for simple traversal, but normal `for` loops when indices are required.

---

# Revision Notes

- A `for` loop repeats code a fixed number of times.
- It has three parts: initialization, condition, and update.
- Initialization executes once.
- Condition is checked before every iteration.
- Update executes after every iteration.
- Enhanced `for` loop simplifies traversal.
- Nested loops are used for matrices and pattern printing.
- `break` exits the loop.
- `continue` skips the current iteration.
- Always watch for off-by-one errors.
